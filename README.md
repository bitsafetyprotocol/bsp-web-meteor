# Starting up

1.  Open Terminal
1.  Install meteor
    ```curl https://install.meteor.com/ | sh```
1.  Clone repo
    ```git clone https://github.com/bitcoinsafetyprotocol/bsp-web-meteor.git```
1.  Run Meteor
    ```cd bsp-web-meteor && meteor run```
1.  Open http://127.0.0.1:3000/ in your browser
